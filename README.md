# Cudobe 🚢

Blockchain-Based Import–Export Management Platform

## 🚀 Overview
Cudobe is a Web3-powered import–export platform that ensures secure trade agreements using Ethereum smart contracts and decentralized file storage with IPFS.

## 🛠 Tech Stack
- Next.js
- PostgreSQL
- Prisma ORM
- Ethereum
- Solidity
- Hardhat
- IPFS

## 📦 Features
- Smart contract–based escrow payments
- Decentralized document storage (IPFS)
- Trade agreement automation
- Shipment tracking system
- Wallet-based authentication
